Horizon Luxury Hotel | Guest Portal
A premium, full-stack Next.js application designed for seamless hotel cabin bookings and guest management. This portal allows guests to reserve cabins, manage their profiles, and view booking history with a high-end UI/UX.

✨ Core Features
Custom Date Selection: A tailored booking calendar built with react-day-picker and date-fns, preventing overlapping bookings.

Authentication: Secure Google OAuth integration via Auth.js (NextAuth).

Dynamic Cabin Management: Real-time availability and price calculations (including discounts).

Guest Dashboard: A dedicated space for guests to edit their profiles and manage existing reservations.

Optimized Performance: Built with Next.js 16+ using Server Components, Turbopack, and Server Actions for snappy interactions.

Responsive UI: Mobile-friendly design featuring a modern sidebar with a custom slim-line scrollbar.

✨Layer,Technology
Framework,Next.js 16 (App Router)
Database,Supabase (PostgreSQL)
Auth,Auth.js (v5)

Dates,date-fns & react-day-picker
